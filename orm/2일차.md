### 1. 터미널 명령어

### 디렉토리 생성
폴더를 생성하고자 하는 경로에 들어가 다음 명령어를 입력한다.

    // mkdir [폴더 이름]
    mkdir dart_projects

    // /user/orm/ 위치에 dart_projects 디렉토리 생성
    mkdir /user/orm/dart_projects


### 디렉토리로 이동

    // cd [이동할 디렉토리 이름]
    cd /home/user/dart_projects

    // 상위 폴더로 이동
    cd ..


### 현재 디렉토리 경로 출력

    // print working directory
    pwd
    

### 터미널 내용 지우기

    clear
    

### Dart 프로젝트 생성

    // dart create -t console [디렉토리 이름]
    dart create -t console cli
